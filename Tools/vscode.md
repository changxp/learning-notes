- 常用快捷键

```
    隐藏资源管理器(即文件目录列表)  
        ctrl + b

    快速打开终端和调试窗口  
        ctrl + ` (Esc键下面那个键)

    选中多行  
        按住Alt键点击左键 多选  或者 Alt+Shift键 鼠标左键上下移动多选

    快速查询 替换
        ctrl + f / ctrl + h  (可以进行正则查询替换)
```
