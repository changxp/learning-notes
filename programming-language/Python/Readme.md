## python3的安装

- 官网 

    [python官网](https://www.python.org/)

- 下载安装包

    wget 

- 解压到指定位置

    tar -xvzf **.tar.gz -C /*/*

- 安装

    ```
    #可以填写参数 编译配置
    ./configure

    #编译
    make && make install
    ```
